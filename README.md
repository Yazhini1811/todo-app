# 🚀 Advanced Todo App

A modern and responsive **Todo Application** built using React with powerful features and a clean UI.

---

## 🌐 Live Demo

🔗 https://darling-valkyrie-6f2ebc.netlify.app/

---

## ✨ Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* 🗑 Delete tasks
* ✔ Mark tasks as completed
* 🌙 Dark Mode toggle
* 💾 Data persistence using LocalStorage
* 🎬 Smooth UI animations
* 📱 Responsive design

---

## 🛠 Tech Stack

* ⚛️ React (Functional Components + Hooks)
* 🎨 CSS3 (Modern UI & Animations)
* 💡 JavaScript (ES6+)

## 🧠 Concepts Used

* useState Hook

* Conditional Rendering

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/Yazhini1811/todo-app.git
```

2. Navigate to project folder

```
cd todo-app
```

3. Install dependencies

```
npm install
```

4. Run the app

```
npm start
```

---

## 🚀 Deployment

This project is deployed using Netlify.

To deploy your own:

```
npm run build
```

Then upload the **build** folder to Netlify.
